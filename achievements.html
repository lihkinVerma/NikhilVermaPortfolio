Top Gun Award (Special Mention)

I worked on parsing resumes from email responses which had challenges of data analysis/feature reduction/clustering. It was a design on a heuristic-based model with a data analysis pipeline where SOTA clustering algorithm was used on a real dataset. Unlike ML-based parser this email parser is a static heuristic parser, constructed using pattern recognition, so its accuracy is 100%. As an impact, this feature has helped to increase the coverage as emails make more than 15% of total responses coming to a system that used to be ignored earlier.
